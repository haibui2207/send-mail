# # Sending email to user through CSV file using Mailgun or Nodemailer

# # Stacks supported
1. Email template
2. Mailgun
3. Nodemailer
4. CSV
5. MongoDb

# # How to run
### 1. Change configs in file `configs.js`

### 2. Install node modules

`npm install`

### 3. Update data in CSV file 

### 4. Start app

`node ./index.js`
